What is Ionic Framework?

Built on top of AngularJS and Apache Cordova (formerly PhoneGap), Ionic provides tools and services for developing hybrid mobile apps using Web technologies like CSS, HTML5, and Sass.

Apps can be built with these Web technologies and then distributed through native app stores to be installed on devices by leveraging Cordova.

The History of Ionic Framework

Ionic was created by Max Lynch, Ben Sperry, and Adam Bradley of Drifty Co. in 2013.

Taking feedback from customers who tried to build mobile apps, the team decided to build their own framework that would focus on performance and be built with modern Web standards, Ionic 1.0 was released in May 2015.

In 2015 Ionic developers reportedly created over 1.3M apps with the SDK

Why use Ionic?

TL;DR: "Ionic's ultimate goal is to make it easier to develop native mobile apps with HTML5, also known as Hybrid apps."

Get started with Ionic Framework


  1. First, install Node.js 4.

  2. Then, install the latest Cordova and Ionic command-line tools. Follow the Android and iOS platform guides to install required platform dependencies.

  3. Create an Ionic project using one of their ready-made app templates, or a blank one to start fresh.

  4. Use the Ionic tool to build, test, and run your apps (or use Cordova directly).

  5. Then, try Ionic View to share your apps with testers and clients, or to easily test on new devices.

  6. Export your application with Cordova.

  7. You will still need to pay for developer privileges to publish to the Apple App Store ($99) or the Google Play Store ($25).
